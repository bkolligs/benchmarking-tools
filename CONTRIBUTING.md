# Contributing 

These guidelines are to make sure style conforms across the project. 

## Naming

| Data                           | Naming Style         |
|--------------------------------|----------------------|
| Variables                      | snake_case           |
| Functions                      | camelBack            |
| Classes                        | CamelCase            |
| Class private member variables | trailing_underscore_ |
